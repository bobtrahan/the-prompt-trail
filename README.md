# The Prompt Trail

> An AI Developer Survival Simulator

Oregon Trail meets AI development. Manage your budget, hardware, and sanity across 13 days of increasingly absurd AI projects. Choose your class, configure your agents, survive the hallucinations.

**Gamedev.js Jam 2026** · Theme: Machines

## Play

🎮 [Play on itch.io](https://bobtrahan.itch.io/the-prompt-trail)
🌐 [Play on GitHub Pages](https://bobtrahan.github.io/the-prompt-trail/)

## Tech

- [Phaser 3](https://phaser.io/) — Game engine
- TypeScript + Vite — Build tooling
- Original concept, art, and audio

## Dev

```bash
cd game
npm install
npm run dev
```
